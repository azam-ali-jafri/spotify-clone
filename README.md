# Spotify Clone

![Img](https://res.cloudinary.com/dg18lu61g/image/upload/v1697531215/projects%20thumbnails/Screenshot_2023-10-17_at_13-56-32_Spotify_clone_ei4szn.png)

### Features:
- Modern and user-friendly UI.
- Supabase as the backend for user authentication and data storage.
- Library creation via music upload.
- Personalized playlist via liked songs.
- Music playback.


## Installation

#### Cloning the repository

```shell
git clone https://github.com/azam-ali-jafri/spotify-clone.git
```

#### Install packages

```shell
npm install or yarn install
```

#### Setup .env file

```js
NEXT_PUBLIC_SUPABASE_URL=
NEXT_PUBLIC_SUPABASE_ANON_KEY=
SUPABASE_SERVICE_ROLE_KEY=
NEXT_PUBLIC_STRIPE_PUBLISHABLE_KEY=
STRIPE_SECRET_KEY=
STRIPE_WEBHOOK_SECRET=

```


#### Start the app
```shell
npm run dev
```
